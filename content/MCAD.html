Siemens NX:

Installation file: SiemensNX-2027.3701_wntx64.zip (need password to download, size 10.7GB)

NX2027.3401_lite_cad2022.7z (need pasword to download, size 2GB, unziped size 8GB)

NX 可攜版的 user default 設定將配合啟動批次檔案中的 set UGII_LOCAL_USER_DEFAULTS=%cd%\NX_user.dpv 將設定資料存入.

因此有關是否參與 Siemens Product Excellence Program 可以經由下列流程進行設定:

File - Utilities - Customer Defaults - Gateway - General - Miscellaneous - Product Excellence Program

選擇是否參加後, 若可攜套件中無前述批次檔案中的 NX_user.dpv 則可透過 Manage current settings 以 Export 將設定存入 NX_user.dpv.

可攜 NX 啟動批次檔案中有超過四百個啟動環境變數設定, 使用者可針對實際需求進行修改後重新啟動.

Win10_2022_nx_web_based_server.7z (need password to download, 33GB)

start_nx.bat (通常認證伺服器使用 port 28000 或 29000)

set SPLM_LICENSE_SERVER=28000@your_license_server_ip_or_domain_name

set UGII_LANG=english

set UGS_LICENSE_BUNDLE=ACD11,ACD10

set ugii=%cd%\ugii
path = %ugii%;%path%

start ugraf -nx
